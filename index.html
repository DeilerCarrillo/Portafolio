Sistema de Gestión de Casos y Citas (Fibra Óptica)

ASP.NET Core · EF Core · SQL Server · Identity · Roles

Problemática: La empresa gestionaba citas y casos de clientes mediante Google Calendar, lo que provocaba falta de control, duplicidad de agendas y dificultad para dar seguimiento.

Solución: El sistema centraliza la gestión de casos y citas, permite asignarlos por planilla/cuadrilla y controla el acceso según el rol del usuario.

Vista actual Tip: click en una imagen para ampliarla

  • Gestión centralizada de casos y citas de clientes.
  • Asignación de trabajos a técnicos o planillas específicas.
  • Control de acceso por roles (Administrador / Técnico).
  • Historial de casos con evidencias y estados.
  • Reportes administrativos para medir la efectividad de las planillas.

Resultado: mayor control operativo, reducción de errores en la agenda y mejor toma de decisiones mediante reportes administrativos.

Diser | Sitio Web Corporativo (Internet)

HTML · CSS

Problemática: La empresa Diser necesitaba una página corporativa para explicar sus servicios de internet, mostrar planes y precios, y ofrecer canales claros de soporte y contacto.

Solución: Sitio multipágina hecho con HTML + CSS, con secciones de inicio, sobre la empresa, servicios, planes/precios, cobertura, galería, FAQ, soporte y contacto.

Vista actual Tip: click en una imagen para ampliarla

  • Diseño limpio y responsive (desktop / mobile).
  • Sección de planes y precios para los servicios.
  • Formularios y opciones para contactar soporte.
  • Galería, avisos y consejos para clientes.
  • FAQ para resolver dudas frecuentes.

Resultado: una presencia online más profesional para Diser y un flujo más claro para que el cliente vea planes, cobertura y pueda solicitar soporte.
