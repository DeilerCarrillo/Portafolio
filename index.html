Proyectos

Sistema de Gestión de Casos y Citas (Fibra Óptica)

ASP.NET Core · EF Core · SQL Server · Identity · Roles

La empresa gestionaba las citas y casos de clientes mediante Google Calendar, lo que provocaba falta de control, duplicidad de agendas y dificultad para dar seguimiento a los trabajos realizados. El sistema centraliza la gestión de casos y citas, permitiendo asignarlos por planilla o cuadrilla y controlar el acceso según el rol del usuario.

  • Gestión centralizada de casos y citas de clientes.
  • Asignación de trabajos a técnicos o planillas específicas.
  • Control de acceso por roles (Administrador / Técnico).
  • Historial de casos con evidencias y estados.
  • Reportes administrativos para medir la efectividad de las planillas.

Resultado: mayor control operativo, reducción de errores en la agenda y mejor toma de decisiones mediante reportes administrativos.
