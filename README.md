# Portafolio — Deiler Carrillo

Portafolio personal enfocado en desarrollo **Backend .NET**.

## Proyecto destacado: Sistema de Gestión de Casos y Citas (Fibra Óptica)

### Problemática

La empresa gestionaba citas y casos de clientes mediante **Google Calendar**, lo que provocaba:

- falta de control operativo,
- duplicidad de agendas,
- poco seguimiento sobre trabajos realizados,
- dificultad para medir rendimiento por planilla/cuadrilla.

### Solución

Se desarrolló un sistema web que centraliza la gestión de casos y citas, con control de acceso por roles.

### Funcionalidades clave

- Gestión centralizada de casos y citas.
- Asignación a técnicos/planillas.
- Control de acceso por roles (Administrador / Técnico).
- Historial por caso con evidencias y estados.
- Reportes administrativos para medir efectividad de planillas.

### Tecnologías

- ASP.NET Core
- Entity Framework Core
- SQL Server
- Identity (Roles)
